This is a set of non-UI components that build on top of JSF to provide additional runtime features. To use these components, simply place the jsf-components.jar in your WEB-INF/lib directory or place it in the shared lib directory of your container. These components use the self configuration mechanisms provided by the JSF specification to automatically install the components into the JSF runtime.

The following components are available in the current release.

  • Ruby on Rails-like Flash concept, exposed via the EL.

    Please see FlashELResolver.

  • PhaseListener for loading non-Java class resources out of a JAR and into a browser, such as images, JavaScript files, and style-sheets.

    Please see ResourcePhaseListener.
